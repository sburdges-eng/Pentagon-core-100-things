# Pentagon-core-100-things

## Multi-PR Merge Workflow

This repository includes tools and documentation for merging multiple pull requests efficiently.

### Quick Start

To merge multiple PRs automatically:

```bash
./merge-prs.sh
```

### Documentation

For detailed information about the merge workflow, see [MERGE_WORKFLOW.md](MERGE_WORKFLOW.md).

### Files

- `merge-prs.sh` - Automated script for merging multiple PRs
- `MERGE_WORKFLOW.md` - Detailed documentation on merge workflows and best practices
